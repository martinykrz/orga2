* ***BLEND*** ***P***acked ***D***ouble float
```asm

```

```asm
blendvps xmm0, xmm1, CTE
```
* ***BLEND*** ***V***ariable ***P***acked ***D***ouble float
```asm
blendvps xmm0, xmm1, CTE
```
