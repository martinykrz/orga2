# Empaquetado
Todos usando saturacion
* ***PACK*** ***S***ign <- ***S***ign, ***D***ouble -> ***W***ord 
```asm
packssdw xmm0, xmm1
```
* ***PACK*** ***U***nsign <- ***S***ign, ***D***ouble -> ***W***ord
```asm
packusdw xmm0, xmm1
```
* ***PACK*** ***S***ign <- ***S***ign, ***W***ord -> ***B***yte
```asm
packsswb xmm0, xmm1
```
* ***PACK*** ***U***nsign <- ***S***ign, ***W***ord -> ***B***yte
```asm
packuswb xmm0, xmm1
```
